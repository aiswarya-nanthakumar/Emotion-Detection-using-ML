# Emotion Detection Using Machine Learning

## Overview
This project detects human emotions from text using Natural Language Processing (NLP) and Machine Learning.

## Features
- Text preprocessing
- TF-IDF Vectorization
- Emotion Classification
- Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- Seaborn
